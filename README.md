# ScriptManager-Py

一个面向开发者的轻量级脚本索引与归档系统。

## 项目介绍

ScriptManager 旨在解决脚本文件随处存放、调用路径繁琐以及缺乏归档的问题。它通过“原地索引 + 异步备份”的逻辑，实现在终端任何位置通过别名快速调用分布在计算机各处的脚本。

### 核心特性

* 原地调用：脚本保留在原始工作目录，完美支持相对路径依赖。
* 自动归档：添加脚本时自动同步副本至 storage 目录。
* 参数透传：支持无损传递参数（如 -h, --config）给原脚本。
* 树状分类：支持多级分类路径（如 药学/CADD/分析）。

## 安装指南

1. 下载 `manager.py` 至本地固定目录。
2. 在 PowerShell 配置文件（$PROFILE）中添加以下函数：

```powershell
function s {
    python "你的路径\manager.py" $args
}

```

## 使用方法

### 1. 添加与归档

将脚本加入管理系统并分类：
`s -add <分类/子类> <脚本文件路径> <备注>`

### 2. 全局调用

在终端任意位置直接执行：
`s <脚本别名> [参数]`

### 3. 名单展示

查看所有已登记的脚本及其备注：
`s -l`

### 4. 其它指令

* 添加/修改分类备注：`s -cat <分类名> <备注>`
* 修改脚本备注：`s -n <脚本名> <新备注>`
* 帮助信息：`s -h`

## 目录结构

* `manager.py`: 系统核心逻辑。
* `data.json`: 索引数据库（自动生成）。
* `storage/`: 归档文件存放目录（自动生成）。
